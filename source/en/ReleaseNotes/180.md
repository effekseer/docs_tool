# 1.80 New Features

## Common

### Collisions

Particles can now collide with surfaces.
Supports external collisions, bounce, friction, lifetime reduction, and coordinate system settings.

### Track, Ribbon

Added settings to repeat UVs at a specified length.

### Rotation

Particles can now rotate to face their movement direction.

### Triggers / Spawning

Added two spawn modes: Continuous and Trigger.
Supports parent-removed and parent-collision triggers, and per-trigger spawn counts.

### External Models

External models can now be registered as emission sources.
During spawning/rendering, you can specify an external model index and coordinate system (local/world).

### Rendering

Added directional billboarding.
Supports UV horizontal flip probability, mirror wrap, tile length, and per-particle tiling.

### Models

Updated the model converter and format support, with loading for `gltf / glb / obj / geo / bgeo`.

### Materials

Added the following nodes to the material editor.
- Smooth Step
- Branch
- Compare
- Logical AND / Logical OR / Logical NOT
- `IsFrontFace`
- ParticleTime
- White Noise / Cellular Noise
- RGB to HSV / HSV to RGB

### GPU Particles

Added GPU-based particle simulation.
Note: This is currently a preview feature and may not be available in all environments.

### Profiler

Added CPU/GPU load measurement and display.
Note: This is currently a preview feature and may not be available in all environments.

## Tools

### Viewer

Ground collision can now be enabled in the viewer.

## Unity

- Updated EffekseerForUnity to the 1.8 series.
- Added a menu to import `.efk` files together with dependent resources.
- Added trigger APIs to EffekseerHandle and EffekseerEmitter.
- Added layer mask support in URP/HDRP custom render passes.
- Added URP RenderGraph support (Unity 6).
- WebGL plugin is selected by Unity version:
  - Unity 2021.2 or later: `2.0.19`
  - Unity 2022.2 or later: `3.1.8`
  - Unity 2023.2 or later: `3.1.38`
