# 1.80 新功能

## 通用

### 碰撞

粒子现在可以与平面发生碰撞。
支持外部碰撞、反弹（Bounce）、摩擦、寿命衰减和坐标系设置。

### 轨迹、丝带

新增按指定长度重复 UV 的设置。

### 旋转

粒子现在可以沿移动方向旋转。

### 触发器/生成

新增两种生成模式：连续与触发器。
支持父节点消失/父节点碰撞触发器，以及每次触发的生成数量设置。

### 外部模型

外部模型现在可以注册为生成源。
在模型生成/渲染时，可指定外部模型索引与坐标系（本地/世界）。

### 渲染

新增移动方向 Billboard。
支持 UV 水平翻转概率、镜像包裹、瓦片长度，以及按粒子瓦片。

### 模型

更新模型转换器与格式支持，新增 `gltf / glb / obj / geo / bgeo` 加载。

### 材质

材质编辑器新增以下节点：
- 平滑步进
- 分支
- 比较
- 逻辑与（AND）/ 逻辑或（OR）/ 逻辑非（NOT）
- 正面判定（`IsFrontFace`）
- 粒子时间
- 白噪声 / 蜂窝噪声
- RGB转HSV / HSV转RGB

### GPU 粒子

新增基于 GPU 的粒子计算功能。
注意：该功能当前为预览版，在部分环境中可能不可用。

### 性能分析器

新增 CPU/GPU 负载的测量与显示功能。
注意：该功能当前为预览版，在部分环境中可能不可用。

## 工具

### 预览器

现在可以在预览器中启用地面碰撞。

## Unity

- 将 EffekseerForUnity 更新至 1.8 系列。
- 新增用于导入 `.efk` 文件及其依赖资源的菜单。
- 为 EffekseerHandle 与 EffekseerEmitter 添加触发器发送 API。
- URP/HDRP 自定义渲染通道支持图层遮罩。
- 新增 URP RenderGraph（Unity 6）支持。
- WebGL 插件会按 Unity 版本自动选择：
  - Unity 2021.2 及以上：`2.0.19`
  - Unity 2022.2 及以上：`3.1.8`
  - Unity 2023.2 及以上：`3.1.38`
