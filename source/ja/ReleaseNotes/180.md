# 1.80 新機能

## 共通

### 衝突

パーティクルを面に衝突させられるようになりました。
外部衝突、バウンス（反発）、摩擦、寿命の減衰、座標系の指定に対応しています。

### 軌跡、リボン

指定した長さごとにUVを繰り返す設定を追加しました。

### 回転

パーティクルを移動方向に合わせて回転できるようになりました。

### トリガー/生成

生成モードとして「連続」と「トリガー」を追加しました。
親消滅時・親衝突時トリガーと、トリガーあたり生成数の設定に対応しています。

### 外部モデル

外部モデルを生成元として登録できるようになりました。
モデルの生成・描画時に、外部モデルのインデックスと座標系（ローカル/ワールド）を指定できます。

### 描画

移動方向ビルボードを追加しました。
UV左右反転確率、ミラーラップ、タイルの長さ、タイル（パーティクル毎）に対応しました。

### モデル

モデルコンバータとフォーマットを更新し、`gltf / glb / obj / geo / bgeo` の読み込みに対応しました。

### マテリアル

マテリアルエディタに以下のノードを追加しました。
- スムースステップ
- 分岐 (Branch)
- 比較 (Compare)
- 論理積 (AND) / 論理和 (OR) / 論理否定 (NOT)
- `IsFrontFace`
- パーティクル時間
- ホワイトノイズ / セルラーノイズ
- RGBからHSV / HSVからRGB

### GPUパーティクル

GPUでパーティクルを計算する機能を追加しました。
※現在はプレビュー機能のため、環境によっては使用できない場合があります。

### プロファイラ

CPU/GPU負荷を計測・表示する機能を追加しました。
※現在はプレビュー機能のため、環境によっては使用できない場合があります。

## ツール

### ビューア

ビューアで地面との衝突を有効化できるようになりました。

## Unity

- EffekseerForUnityを1.8系に更新しました。
- .efkファイルと依存リソースをまとめてインポートするメニューを追加しました。
- EffekseerHandleとEffekseerEmitterにトリガー送信APIを追加しました。
- URP/HDRPのカスタムレンダーパスでレイヤーマスク指定に対応しました。
- URP RenderGraph（Unity 6）に対応しました。
- WebGLプラグインはUnityバージョンに応じて以下を使用します。
  - Unity 2021.2以降: `2.0.19`
  - Unity 2022.2以降: `3.1.8`
  - Unity 2023.2以降: `3.1.38`
